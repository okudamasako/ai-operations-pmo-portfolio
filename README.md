# AI Operations / PMO Portfolio

AI・自動化・PMO運用をテーマにしたポートフォリオサイトです。

## Overview

業務整理・AI活用・自動化設計を中心に、
実際の管理画面や運用フローを掲載しています。

* AI Workflow
* PMO / 業務整理
* Notion Database運用
* GitHub / Vercel管理
* AIニュース整理ツール
* Markdown生成支援

## Tech Stack

* HTML
* Tailwind CSS
* Vercel
* GitHub

## Features

* ポートフォリオ公開
* 実運用画面掲載
* AIワークフロー整理
* 業務構造の可視化
* スマホ対応
